前言

   看到網路上很多人都有自己的個人網頁， 剛好寒假有空所以就決定來做一個屬於自己的網頁， 但因為沒什麼美感所以做得很簡陋QQ

心得

   如果想做一個屬於自己的網頁，這邊建議直接用 Hexo 來做， 直接套現有模板比自己慢慢排版跟加功能快多了。
   因為我不太會javascript的關係，常常寫到一半就要上網查相關語法， 但其實現在有ChatGPT的幫助下其實也做的不算慢， 在寫完網頁的框架之後我就用 Github Page 來把我的網頁放到網路上了， 整體來說架一個網頁其實很簡單。

總結

   如果對 HTML, CSS, Javascript 都很熟而且勵志要當前端工程師的話可以跟我一樣自己手刻網頁， 除此之外的人還是用網路上的模板就好><
